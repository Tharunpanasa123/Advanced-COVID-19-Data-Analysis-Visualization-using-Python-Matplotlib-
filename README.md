
# Advanced COVID-19 Data Analysis & Visualization

## Project Overview
This project performs advanced COVID-19 data analysis using Python.

### Key Features:
- Exploratory Data Analysis (EDA)
- Daily Growth Rate Calculation
- Case Fatality Rate (CFR) Computation
- Line, Bar, and Pie Chart Visualizations

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the analysis:
   python main.py

Charts will be generated in the project directory.
